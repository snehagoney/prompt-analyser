# Prompt Analyzer

A small Python project that analyzes the quality of a prompt and gives simple feedback.

## What this project does

This project checks whether a prompt is:

- too short
- missing clarity
- missing action words
- missing context

## Project Structure

```text
prompt-analyzer/
│
├── src/
│   ├── main.py
│   └── analyzer.py
│
├── .gitignore
├── README.md
└── requirements.txt
```

## How to run

### On Mac

```bash
cd src
python3 main.py
```

### On Windows

```bash
cd src
python main.py
```
